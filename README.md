# dotfiles

## usage

### https

```sh
$ git clone https://github.com/nakahiro386/dotfiles.git ~/repo/github.com/nakahiro386/dotfiles
$ cd !$
$ ./init.sh --dry-run
$ ./init.sh

$ git clone https://github.com/nakahiro386/vimfiles.git ~/repo/github.com/nakahiro386/vimfiles
$ ln -s ~/repo/github.com/nakahiro386/vimfiles ~/.vim
```

### ssh

```sh
$ git clone git@github.com:nakahiro386/dotfiles.git ~/repo/github.com/nakahiro386/dotfiles
$ cd !$
$ ./init.sh --dry-run
$ ./init.sh

$ git clone git@github.com:nakahiro386/vimfiles.git ~/repo/github.com/nakahiro386/vimfiles
$ ln -s ~/repo/github.com/nakahiro386/vimfiles ~/.vim
```
